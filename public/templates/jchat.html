jChat (or jChatV2) is the second version of my client/server chat application.

It was created mainly to facilitate the communication between my classmates during those long and boring classes. Some of it's features include:
  • IP/port configuration
  • Unlimited number of clients
  • Client username, themes, font type, size and color
  • Connected clients list
  • Kick function and "panic button" from the server application
  • On screen logs
