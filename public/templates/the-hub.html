TheHub is my first big web project. It's a chat room web application that supports dynamic room creation and features such as:
  • User profiles with basic information and avatar
  • User panel to edit all user information
  • Drag and drop for all interface panels
  • Creation of public, private (password protected), persistent or temporal chat rooms with user limits
  • Automatic deletion of inactive/empty chat rooms
  • Emoticons and quick view of the users profiles while chatting
  • Admin panel with room creation and deletion options, and kick and ban user functionalities
